# Fantastic UI 更新日志

## v1.2.0

**发布日期：** 2023-06-15

### 新增组件

- **Rate 评分组件**
  - 支持自定义评分数量、图标和颜色
  - 支持自定义图标大小和间距
  - 支持半星、只读和禁用状态
  - 支持点击和滑动手势操作

## v1.1.0

**发布日期：** 2023-05-29

### 新增组件

- **Uploader 文件上传组件**
  - 支持图片和文件上传
  - 支持预览、删除、进度显示
  - 支持自定义上传方法
  - 提供多种尺寸和状态

### 优化

- 优化组件文档结构
- 完善示例页面
