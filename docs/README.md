# Fantastic UI 更新日志

## v1.33.0

**发布日期：** 2026-01-15

### 新增组件

- **Video 视频播放器组件**
  - 用于视频文件的播放控制和展示，提供多种样式和功能选项
  - 支持多种类型：默认、主要、成功、警告、危险和信息
  - 支持播放/暂停、进度控制、音量调节等基础功能
  - 支持封面图片展示和中央播放按钮
  - 支持全屏播放和画中画模式（如果浏览器支持）
  - 支持禁用状态、自动播放和循环播放
  - 支持多种视频填充模式：包含、覆盖和填充
  - 支持显示缓冲进度和加载状态
  - 提供丰富的事件回调，方便业务逻辑处理
  - 适用于视频播放、教学视频、产品展示等多种场景
