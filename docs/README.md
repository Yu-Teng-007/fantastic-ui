# Fantastic UI 更新日志

## v1.66.0

**发布日期：** 2025-09-17

### 新增组件

- **fanc-highlight 文字高亮组件**
  - 用于突出显示文本中的关键词或搜索结果
  - 支持自定义高亮颜色和背景色
  - 支持多个关键词同时高亮显示
  - 支持大小写敏感和不敏感模式
  - 提供匹配全词和部分匹配两种模式
  - 支持自定义高亮样式，包括文字颜色、背景颜色、字体粗细等
  - 适用于搜索结果展示、关键信息强调等场景
